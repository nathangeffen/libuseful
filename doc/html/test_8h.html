Detailed Description

Declare a test_group struct and initialize it with test_group_new(). Use the macros TEST, TESTEQ, TESTGT and TESTLT to run tests. The function test_group_summary() prints the number of successes and failures in your test group. To reduce the amount of output to stdout set the test_group's verbose value to false.
