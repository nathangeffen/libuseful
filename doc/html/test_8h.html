Detailed Description

Declare a u_test_group struct and initialize it with u_test_group_new(). Use the macros U_TEST, U_TESTEQ, U_TESTGT and U_TESTLT to run tests. The function u_test_group_summary() prints the number of successes and failures in your test group. To reduce the amount of output to stdout set the u_test_group's verbose value to false.
