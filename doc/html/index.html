Useful: Some algorithms, macros and utilities for C programmers

Introduction

This is a collection of useful C algorithms, macros and utilities.

It includes:

  • Testing framework (a very simple one): test.h
  • Array management macros and functions: array.h
  • Algorithms that are not already in glib or gsl: algorithms.h
  • CSV file input and output, and conversion to and from R-like dataframes: csv.h
  • String handling functions that handle memory memory allocationallocationMacros to do make C error checking a touch more convenient @section install_sec Installation Install libuseful using a package manager. To use it Useful in your program, include <useful.h> in your source code. Alternately include just what you need, i.e. <useful/test.h>, <useful/array.h> and/or <useful/algorithms.h>. The easiest way to compile is to use pkg-config.
