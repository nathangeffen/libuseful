Detailed Description

Most of the time you will want to use the csv_read() function to read a CSV file into a 2-d array structure of strings (struct csv). Then when you're done with it, call csv_free().

To create a csv structure and then csv files use csv_new(), csv_append() and csv_write().

Use csv_at() to get the string value of a particular cell. Use csv_to_matrix() to convert a csv consisting of numbers to a matrix of doubles. Remember to free your matrix when done with it using matrix_free().

Everything else provided here is just cute, or perhaps occasionally useful.
